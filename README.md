# Roshambo

## Description
This is a browser-based Rock, Paper, Scissors game built using JavaScript. 
The game uses prompt(), alert(), and confirm() to interact with the user.

## How to Play
- Enter R, P, or S when prompted
- The computer randomly selects its move
- Wins, losses, and ties are tracked
- Choose whether to play again after each round

## Technologies Used
- HTML
- JavaScript
- CSS
